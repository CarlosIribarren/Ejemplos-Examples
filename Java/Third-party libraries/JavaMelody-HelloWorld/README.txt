Se puede colocar el war de "JavaMelody-HelloWorld" dentro de un tomcat con mas aplicaciones (wars).
Al entrar en la pantalla de monitoreo se mostarar un estado general del servidor.

Compilar el war de "JavaMelody-HelloWorld" y arrancar el Tomcat con el resto de wars.

Entrar en el pantalla de monitoreo:
-----------------------------------

	http://localhost/JavaMelody-HelloWorld/monitoring
